# YieldBasis Stablecoin Derivatives

YieldBasis Stablecoin Derivatives is a spec-first idea for building cross-protocol stable tranche vaults anchored by YieldBasis Hybrid Vaults and veYB-linked incentive rails.

Interpretation note:
- this pack interprets the user's "trans faults" phrasing as a mix of:
  - cross-protocol fault domains
  - cross-time tranche and lock fault domains
