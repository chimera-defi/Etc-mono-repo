## Interaction Patterns

### Voice Button States
1. **Idle:** Pulsing blue circle
2. **Listening:** Red recording indicator + waveform
3. **Processing:** Spinner animation
4. **Speaking:** Sound wave animation
5. **Error:** Red shake animation

### Haptic Feedback
- **Voice button tap:** Light impact
- **Command recognized:** Success notification
- **Agent created:** Success notification
- **Error:** Error notification

### Animations
- **Screen transitions:** Slide from right (300ms)
- **Modal present:** Slide from bottom (250ms)
- **Voice button:** Scale + fade (200ms)
- **List items:** Fade in + slide up (150ms, staggered)

## Accessibility

### Voice Over Support
- All buttons have descriptive labels
- Status indicators announce changes
- Form fields have clear labels
- Navigation is fully keyboard accessible

### Text Scaling
- Supports Dynamic Type (iOS)
- Supports system font size (Android)
- Layouts adapt to larger text

### Color Contrast
- All text meets WCAG AA standards (4.5:1 minimum)
- Primary actions have 7:1 contrast
- Icons have clear visual indicators beyond color
