## Competitor Matrix (Intelligence Exchange)

Last updated: 2026-03-07

Scoring legend:
- `Strong`: first-class capability
- `Partial`: capability exists but not core or limited
- `Weak`: mostly absent

| Competitor | Category | Task Market Modes | Trust/Reputation Layer | Settlement/Dispute Focus | Agent Protocol Surface | Notes |
|---|---|---|---|---|---|---|
| Daydreams (Taskmarket + Router) | Agent commerce network | Strong | Strong | Partial | Strong | Closest conceptual competitor for agent-first execution market. |
| OpenRouter | Model/API router | Weak | Weak | Weak | Partial | Strong routing abstraction and BYOK; limited marketplace settlement semantics. |
| Together AI | Inference + GPU cloud | Weak | Weak | Weak | Partial | Strong infra/performance; not a task outcome marketplace. |
| Replicate | Model execution API platform | Weak | Weak | Weak | Partial | Strong model execution surface; limited worker-market design. |
| Vast.ai | Compute marketplace | Partial | Weak | Weak | Weak | Supply depth for raw compute; high operational burden for buyers. |
| TensorDock | Compute marketplace | Partial | Weak | Weak | Weak | Similar to Vast.ai category; infra-centric rather than task-settlement-centric. |
