## Orbit Pilot Validation Plan

### Goal
Validate that Orbit Pilot saves meaningful launch-ops time while staying inside a conservative compliance posture.

### Phase 0
1. Interview 10 launch operators or founders.
2. Gather 3 real launch checklists and directory lists.
3. Rank platforms by actual usage frequency and friction.

### Phase 1
1. Load one real launch profile.
2. Build 20-platform seed registry.
3. Measure time to create all drafts and manual packages.

### Phase 2
1. Add 3-5 official publishers.
2. Run live campaigns on one real launch.
3. Measure tracking quality and submission completion.

### Success Thresholds
1. >= 50% reduction in submission-prep time.
2. >= 95% of outbound links correctly tagged.
3. zero policy incidents in pilot.
4. >= 80% of target platforms receive a usable draft or publish action.

### Kill Thresholds
1. maintenance burden dominated by brittle browser automation,
2. insufficient demand for structured launch ops,
3. users reject manual-first positioning,
4. directory ROI is too weak to justify product depth.
