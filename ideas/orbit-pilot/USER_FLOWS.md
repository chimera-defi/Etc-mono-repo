## Orbit Pilot User Flows

### Flow 1: New Launch
1. Create or import launch profile.
2. Select target platform bundle.
3. Review generated drafts and asset variants.
4. Run launch.
5. Review results and manual queue.

### Flow 2: Manual Submission
1. Open queued platform.
2. Copy approved draft and asset pack.
3. Submit manually.
4. Record live URL and notes.

### Flow 3: Registry Update
1. Open platform record.
2. Change mode/risk/URL constraints.
3. Save and re-run only affected targets.
