## Orbit Pilot Financial Model

### Model
Operator SaaS with per-workspace subscription plus usage-based add-ons.

### Variables
- `W` = active paying workspaces
- `P` = average monthly subscription
- `U` = usage add-on revenue
- `C_ai` = variable AI/content cost
- `C_ops` = support + infra cost

### Formula
- `MRR = W * (P + U)`
- `COGS = W * (C_ai + C_ops)`

### Example Base Case
- `W = 150`
- `P = $49`
- `U = $21`
- `C_ai = $8`
- `C_ops = $10`
- `MRR = $10,500`
- `COGS = $2,700`
- `Gross Margin ≈ 74.3%`

### Why This Can Work
The product is operational leverage software. If it meaningfully compresses launch-prep time, the pricing ceiling is higher than a simple copy generator.
