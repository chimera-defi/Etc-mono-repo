# SpecForge Pricing Benchmarks

Updated: 2026-03-19

Public benchmarks used for the current SpecForge pricing framing:

1. Notion pricing:
   - Plus: `$10` per member / month
   - Business: `$20` per member / month
   - Source: https://www.notion.com/pricing

2. Linear pricing:
   - Basic: `$10` per user / month
   - Business: `$16` per user / month
   - Source: https://linear.app/pricing

3. Confluence pricing:
   - Premium: `$10.44` per user / month
   - Enterprise: contact sales
   - Source: https://www.atlassian.com/en/software/confluence/pricing

How SpecForge uses these:

1. The current `Team SaaS` price is positioned above basic documentation/collaboration tools because it includes multiplayer authoring, governed agent review, and launch-packet delivery.
2. The current page is still a positioning surface, not a live billing contract.
3. Final packaging should be validated against design-partner usage, not set purely from competitor anchors.
