# SpecForge Pricing Benchmarks

Updated: 2026-03-20

Public benchmarks used for the current SpecForge pricing framing:

1. Notion pricing:
   - Plus: `€9.50` per member / month on the currently served page
   - Business: `€19.50` per member / month on the currently served page
   - Custom Agents Beta: `$10 per 1,000 credits`
   - Source: https://www.notion.com/pricing

2. Linear pricing:
   - Basic: `$10` per user / month
   - Business: `$16` per user / month
   - Source: https://linear.app/pricing

3. Cursor pricing:
   - Teams: `$40` per user / month
   - Enterprise: custom
   - Source: https://cursor.com/pricing

4. Coda AI credits:
   - Pro: `2,000` credits per Doc Maker / month
   - Team: `6,000` credits per Doc Maker / month
   - Unlimited AI add-on: `$12` per Doc Maker / month
   - Source: https://help.coda.io/hc/en-us/articles/39555859321357-Coda-AI-credits

How SpecForge uses these:

1. The current `pilot` rehearsal pricing is closer to a collaboration/workflow seat than a pure docs seat because it combines multiplayer authoring, governed agent review, and execution handoff.
2. The nearest public analogs split into two models:
   - seat-based collaboration pricing (`Linear`, `Cursor Teams`, `Notion Business`)
   - usage/credit overlays for agent work (`Coda AI`, `Notion custom agents`)
3. SpecForge should likely keep the same split:
   - seat-based pilot access
   - usage-based or pooled agent credits on top
4. The current page is still a rehearsal surface, not a live billing contract.
5. Final packaging should be validated against design-partner usage, not set purely from competitor anchors.
