# Positioning Brief

## Category Statement

AI-native collaborative spec workflow platform.

## Primary User

Product and engineering teams shipping software with human + agent collaboration.

## Core Promise

Turn rough product intent into implementation-ready spec artifacts with governed edits, auditability, and exportable handoff packs.

## Differentiator

Shared markdown canvas with patch-governance, readiness gates, and deterministic spec-to-execution outputs across web, API, and terminal flows.

## Scope Boundaries

- In scope: PRD/SPEC/TASKS workflows, collaborative editing, traceability, export/handoff.
- Out of scope: generic office-suite document editing and broad knowledge-base replacement.
