# Frontend Vision

The frontend should feel like a high-conviction protocol war room, not a generic dashboard.

## Visual Direction

- cream or sand background with dark graphite panels
- green and brass accents for productive capital
- warning orange for peg stress and policy changes
- diagrams and flow rails instead of card soup

## Key Screens

1. Landing / thesis
2. Vault and mint dashboard
3. Tranche comparison
4. Agent action console
5. Prize mapping / architecture explainer

## Motion

- Animate capital movement between sources
- Animate peg stress events as state transitions, not confetti
- Stagger logs when the agent acts
