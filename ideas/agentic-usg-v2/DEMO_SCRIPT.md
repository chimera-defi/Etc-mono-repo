# Demo Script

## Goal

Deliver a 3-5 minute judge demo with no hidden state and no dependency on perfect live markets.

## Sequence

1. Show thesis screen:
   - productive stablecoin
   - human-backed agent
   - prediction-yield rotation
2. User proves humanness.
3. User deposits collateral and mints `aUSG`.
4. Show current tranche and health factor.
5. Agent detects better prediction-market yield.
6. User approves the sensitive rotation.
7. System executes and shows before/after yield and audit log.
8. Trigger mild peg stress fixture.
9. Show one policy response, not a whole emergency playbook.
10. End on prize mapping and why the architecture fits.

## Hard Rules

- Never imply production-ready peg guarantees.
- Never hide whether a feed is simulated.
- Never show more than one hero agent action in the main path.
