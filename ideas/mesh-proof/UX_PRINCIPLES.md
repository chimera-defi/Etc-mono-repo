# UX Principles

1. **Trust through transparency:** Every provider shows its staked attestation, model hash, and quality score onchain. No black boxes.
2. **Pay for what you use:** Per-job pricing is visible before the prompt is sent. No surprise bills.
3. **Fail fast, recover fast:** If a provider is slow or drops, auto-failover is automatic and visible.
4. **Provider dignity:** The provider dashboard treats inference providers as first-class users, not commoditized endpoints. Earnings, rank, and quality history are always visible.
5. **Governance is opt-in:** veVIEM locking is a power-user feature. Default buyer/provider UX never mentions it.
6. **No KYC, no friction:** Wallet connection is the only gate. No email, no phone, no identity.
7. **Dark by default:** The interface assumes developers and power users. Light mode is secondary.
8. **Keyboard-first:** Power users can run the entire flow from the CLI without touching the mouse.
