## Category Definition

**Stablecoin Banking Services:** Mobile-first platforms that enable P2P payments, remittances, and bank withdrawals using stablecoins/crypto, bridging crypto and traditional banking systems.

**Required Features:**
1. Stablecoin/crypto support
2. **Bank withdrawal (off-ramp)** - REQUIRED
3. P2P payments
4. Mobile apps
5. Regional focus

**Optional Features (Differentiators):**
- Debit cards
- Exchange features
- DeFi features
