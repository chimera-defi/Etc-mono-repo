# Hardware Wallet Setup Guide (Step-by-Step)

## Quick Checklist

1. Buy from the official vendor only.  
2. Update firmware on first boot.  
3. Generate a new seed on-device (never import old seeds).  
4. Write the seed offline; never photograph it.  
5. Enable a passphrase if supported.  

---

## Step-by-Step Setup

### Step 1: Verify Device Integrity
- Check tamper seals (if provided).
- Confirm serial and authenticity via vendor tools.

### Step 2: Initialize On-Device
- Generate a new seed on the device.
- Record the seed offline (paper or metal backup).

### Step 3: Configure Security
- Set a PIN.
- Enable passphrase (recommended).

### Step 4: Test Recovery
- Perform a recovery test using the recorded seed.

### Step 5: Transfer Funds
- Send a small test transaction before large transfers.

---

## FAQ

**Should I import an old seed?**  
No. Always generate a fresh seed on the device.

**Do I need a passphrase?**  
Recommended for stronger protection.
