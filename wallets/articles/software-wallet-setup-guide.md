# Software Wallet Setup Guide (Step-by-Step)

## Quick Checklist

1. Download from the official site or app store.  
2. Create a new wallet and secure the seed.  
3. Enable biometrics/lock screen.  
4. Verify network settings before connecting dApps.  
5. Start with small test transactions.  

---

## Step-by-Step Setup

### Step 1: Install from Official Source
- Use vendor website or verified app store listing.

### Step 2: Create a New Wallet
- Generate a new seed.
- Store it offline (no screenshots or cloud notes).

### Step 3: Lock Down Access
- Enable biometric lock or strong PIN.
- Turn on phishing/scam warnings if available.

### Step 4: Connect to dApps
- Verify the dApp URL.
- Check approvals before signing.

### Step 5: Test First
- Send a small test transaction.
- Confirm you can recover using the seed.

## FAQ

**Is a software wallet safe?**  
It is safe for daily use if you follow good security practices.

**Should I use multiple wallets?**  
Yes, separate daily funds from long-term storage.
