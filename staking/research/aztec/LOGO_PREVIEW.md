## Usage Guidelines

### File Format
- **Format:** SVG (Scalable Vector Graphics)
- **Advantages:** Scalable, web-friendly, print-ready
- **Can be exported to:** PNG, PDF, EPS for various uses

### Size Variations Needed
- **Favicon:** 32x32px, 64x64px
- **Social Media:** 512x512px (Twitter), 1024x1024px (Discord)
- **Header:** 200x200px, 400x400px
- **Print:** High-resolution (300 DPI)

### Color Variations Needed
- **Full Color:** As designed
- **Single Color:** Black or white version
- **Dark Mode:** Inverted colors for dark backgrounds
- **Grayscale:** For single-color printing
